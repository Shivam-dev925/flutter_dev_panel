## 0.1.0

* Initial release of Flutter Inspector (formerly flutter_dev_panel)
* Network Inspector - Intercept and inspect HTTP/HTTPS requests with Dio and http package support
* Storage Viewer - View and edit SharedPreferences in real-time
* Log Viewer - Auto-capture debugPrint() calls and custom logs with different severity levels
* Feature Flags - Toggle features on/off without rebuilding the app
* Quick Actions - Execute custom actions with one tap
* App Info - Display app version, device info, and platform details
* Shake Detection - Open panel by shaking device (iOS/Android)
* Beautiful dark-themed UI with smooth animations
* Production-safe with built-in guards to prevent accidental release exposure
* Full cross-platform support (Android, iOS, Web, macOS, Linux, Windows)
